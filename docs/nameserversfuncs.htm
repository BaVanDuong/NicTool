Functions for Nameservers

  • delete_nameserver Delete a certain nameserver.

  • edit_nameserver Modify the details of an existing nameserver.

  • get_group_nameservers Get a list of the nameservers in a group.

  • get_nameserver Get the details for a certain nameserver.

  • get_nameserver_list Get the details for a list of nameservers.

  • get_nameserver_tree Get the list of nameservers available along the path from the root to a certain group.

  • move_nameservers Move a list of nameservers to a certain group.

  • new_nameserver Create a new nameserver.
