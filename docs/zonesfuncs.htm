Functions for Zones

  • delete_zones Delete a list of zones.

  • edit_zone Modify an existing Zone.

  • get_group_zones Get a list of the zones in a group.

  • get_group_zones_log Get a list of log entries for actions performed on zones in a group.

  • get_group_zone_query_log Get a list of log entries for queries made on zones in a group.

  • get_zone Get the details for a certain Zone.

  • get_zone_list Get the details for a list of zones.

  • get_zone_log Get the log entries for actions on a certain Zone.

  • get_zone_records Get the Zone records in a specific Zone.

  • move_zones Move a list of zones to a certain group.

  • new_zone Create a new Zone.
