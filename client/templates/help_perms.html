Edit Properties Create New Object Delete Object Delegate Object
Group: Edit
The user may edit sub-group properties.		 Create
The user may create new sub-groups.		 Delete
The user may delete sub-groups.		  
User: Edit
The user may edit other users' properties.		 Create
The user may create new users.		 Delete
The user may delete other users.		  
Zone: Edit
The user may edit zone properties.		 Create
The user may create new zones.		 Delete
The user may delete zones.		 Delegate
The user may delegate zones.
Zonerecord: Edit
The user may edit zone record properties.		 Create
The user may create zone records inside zones.		 Delete
The user may delete zone records.		 Delegate
The user may delegate zone records.
Nameserver: Edit
The user may edit nameserver properties.		 Create
The user may create nameservers.		 Delete
The user may delete nameservers.		  
Self: Edit
The user may edit his/her own properties.		      
Special cases: A user may never delete his/her self. A user may never edit his/her own group.
